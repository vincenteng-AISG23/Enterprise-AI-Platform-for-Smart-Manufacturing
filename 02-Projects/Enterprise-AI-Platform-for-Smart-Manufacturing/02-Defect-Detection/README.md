# AI Defect Detection for Smart Manufacturing

## Executive Summary
This project demonstrates how AI can automate visual quality inspection by classifying products as defective or non-defective.

## Business Problem
Manual inspection in manufacturing is:
- Inconsistent
- Slow
- Labor-intensive

This leads to:
- Quality defects reaching customers
- High rework and scrap costs
- Inefficient production processes

## Solution
This project uses machine learning to classify defects based on inspection features such as:
- Edge roughness
- Color variation
- Scratch score
- Shape deviation

## Business Impact
- Improved product quality
- Faster inspection process
- Reduced scrap and rework
- Scalable quality control
