# AI-Driven Smart Manufacturing Control Tower

## Overview
An enterprise AI platform integrating Predictive Maintenance, Defect Detection, and Demand Forecasting into a unified decision-support system.

## Why This Matters
Manufacturing systems often operate in silos:
- Maintenance (downtime)
- Quality (defects)
- Planning (forecasting)

This project demonstrates how AI can unify these into a single control tower.

## Key Capabilities
- Predictive Maintenance (failure prediction + risk scoring)
- Defect Detection (quality analytics + inspection insights)
- Demand Forecasting (trend + accuracy + bias analysis)

## Executive Control Tower
- Unified KPI dashboard
- Financial impact modeling (SGD risk exposure)
- Cross-module AI insights
- Actionable recommendations

## Business Impact
- Reduce downtime
- Improve product quality
- Optimize production planning
- Enable data-driven decisions

## Technology Stack
- Python
- Streamlit
- Scikit-learn
- Pandas / Matplotlib
